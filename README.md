## Pacotes para instalar

- Para ser instalado via CRAN:

``` r
pacotes <-
  c("tidyverse",
    "tidytext",
    "stopwords",
    "wordcloud2",
    "janitor",
    "remotes",
    "readtext",
    "fs")
install.packages(pacotes)
```

- Para ser instalado via GitHub:

``` r
remotes::install_github("dfalbel/rslp")
```
